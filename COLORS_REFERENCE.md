## Using These Colors Externally

If you want to use these themes in other projects or share:

### CSS Copy-Paste
```css
/* Blue Cyan Theme */
:root {
    --primary-light: #1e3a8a;
    --secondary-color: #3b82f6;
    --accent-color: #0ea5e9;
}

/* Purple Pink Theme */
:root {
    --primary-light: #7c3aed;
    --secondary-color: #a855f7;
    --accent-color: #ec4899;
}

/* Add more themes as needed */
```

### SASS Variables
```scss
$blue-cyan: (#1e3a8a, #3b82f6, #0ea5e9);
$purple-pink: (#7c3aed, #a855f7, #ec4899);
$green-teal: (#059669, #10b981, #14b8a6);
// ... etc
```

### Tailwind Config
```js
colors: {
    'blue-cyan': {
        primary: '#1e3a8a',
        secondary: '#3b82f6',
        accent: '#0ea5e9',
    },
    // ... add others
}
```
